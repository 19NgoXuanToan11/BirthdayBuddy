- Sau khi chỉnh sửa, code thì commit và push lên branch của mình sau đó mới preview và tạo pull request

- Nhớ để description ở mỗi lần commit 
- BE thì chỉ thao tác bên folder Backend 
- FE chỉ thao tác bên Frontend
